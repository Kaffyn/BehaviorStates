# Editor Workbench

Ferramentas visuais para produtividade.

## Library

O "Grimório" de Assets. Navegue por todas as habilidades e efeitos do projeto em uma lista filtrável e categorizada.

## Factory

Wizard para criar novos Resources rapidamente. Evita erros manuais e configura defaults sãos.

## Debugger

Ferramenta essencial para desenvolvimento:

- Veja Tags ativas em tempo real.
- Inspecione valores de atributos (Base e Current).
- Veja cooldowns ativos.
